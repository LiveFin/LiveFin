# LiveFin

LiveFin is a Live TV focused client for streaming your TV channels from your personal Jellyfin server onto your iOS, iPadOS, and macOS devices.

**NOTE:LiveFin is not part of the Jellyfin Project and doesn’t host any servers on its platforms**


## Features

- Stream your Live TV channels provided by your Jellyfin server
- Integrated your EPG, which shows data up to however long the data can get
- Minimalistic login page
- Channel images included
- Channel listed numerically (based on tuner config)
- Picture-in-Picture
- Library Support
- Apple Watch Companion
